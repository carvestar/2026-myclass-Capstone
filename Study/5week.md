## 5) 대화 질문의 구성 요소 탐구
### 주요 문자 기호
- 하이픈(-), 목록 작성
  - 목록의 각 항목
- 따옴표("")
  - 특정 단어 문구 강조
  - 모델이 정확하게 인식해야 하는 텍스트 표시
- 별표(*)
  - 목록
  - 특정 내용 강조
- 대괄호[]
  - 추가 정보
  - 선택 사항
- 중괄호{}
  - 특정 조건 명시
- 세 개의 백틱(```)을 앞뒤로 사용
  - 코드 블록
  - 프롬프트 내에서 코드나 명령어를 명확하게 구분
  - (```)python -> 파이썬
  - (```)sql -> sql
- 해시 기호(#)
  - 마크다운 문법에서 제목이나 섹션 구분
  - 프롬프트를 구조화하여 모델이 각 부분을 명확하게 이해하도록 도움
### 대화 질문 설계 요소
- 키워드와 제한
  - 핵심 키워드를 정확하게 사용하는 것 중요
    - Ex1) 친구에게 맛집을 물어볼 때 "매운 떡볶이 맛집"이라고 구체적으로 말하는 것
    - Ex2) "한국 역사"보다는 "조선시대 세종대왕 업적"이라고 질문
  - 답변의 범위나 형식 제한하는 것 유용
    - "100자 이내로 요약해 줘."또는 "표 형식으로 정리해 줘."와 같은 조건
- 어조와 부정
  - 사용자가 요청 시 AI가 더 부드럽고 친절한 어조로 답변할 가능성 높음
    - Ex) "이 주제에 대해 설명해 줘." -> "이 주제에 대해 친절하게 설명해 줘."
  - 부정적인 표현을 사용하여 원치 않는 결과 제외
    - Ex) "인공지능의 장점만 알려 줘." -> "인공지능의 단점은 제외하고 장점만 알려 줘."
### 대화 질문 설계의 10가지 구성 요소

| 유형 (Type) | 설명 (Description) | 예시 (Example) |
| :--- | :--- | :--- |
| **목적 (Objective)** | 글의 목적을 명확히 설정 | "전기차 구매 시 고려해야 할 사항을 알려줘" |
| **상황 (Context)** | 배경 정보, 상황, 추가 자료 설명 | "이 상황의 배경, 목표, 고려해야 할 점, 제약 사항을<br>포함해 설명해줘" |
| **역할 (Role, Persona)** | ChatGPT의 역할 또는 관점을 지정 | "너는 대학 교수야. 마케팅 전문가라 생각하고 답해줘" |
| **청중 (Audience)** | 대상이 되는 목표 독자를 구체적으로 명시 | "초등학교 5학년 수준, 대학 신입생, 경영진 대상" |
| **예시 (Examples)** | 원하는 형식이나 스타일의 예시 제공 | "‘긴 글 → 핵심 요약’ 형태로 만들어줘" |
| **형식 (Format)** | 결과물의 구조나 형식 지정 | "문단으로 정리해줘, 표로 정리해줘" |
| **키워드 (Keywords)** | 반드시 포함해야 할 단어나 개념 지정 | "반복적 접근, 신속한 엔지니어링 포함해줘" |
| **제한 (Limitations)** | 답변의 길이, 단어 수, 문장 수 등의<br>제한 사항을 설정 | "200단어 이내, 3문장으로 작성, 예시 5개 포함" |
| **어조 (Tone)** | 글의 분위기와 스타일 지정 | "친절하게 설명해줘, 전문가의 시각에서 작성해줘" |
| **부정 (Negation)** | 피해야 할 내용 명시 | "기술 용어를 남발하지 말고, 너무 어렵게 설명하지 마" |

## 6) 효율적인 대화 질문: 퓨샷 프롬프팅
### 퓨샷 러닝(Few-shot learning)과 퓨샷 프롬프팅(Few-shot-prompting)

- **퓨샷 러닝(FSL: Few-shot learning)**
  - AI 모델이 제한된 수의 예시만으로 새로운 개념이나 작업 학습
    - 어린 아이가 몇 번의 경험으로 새로운 단어나 사물을 이해하는 방식과 유사
    - 적은 데이터로 높은 학습 효율을 보임
  - 기존의 머신 러닝 모델은 방대한 양의 데이터를 필요로 했지만 퓨샷 러닝은 단 몇 장의 고양이 사진만으로도 새로운 고양이 인식
    - 데이터 부족
      - 새로운 분야나 희귀한 데이터의 경우 충분한 학습 데이터 확보 어려움
      - 퓨샷 러닝은 이러한 상황에서 뛰어난 성능 발휘
    - 빠른 적응
      - 새로운 환경이나 작업에 신속하게 적응해야 할 때
      - 퓨샷 러닝은 적은 데이터로도 학습 가능
    - 개인화
      - 개인의 취향이나 특성에 맞춰 모델 학습시킬 때
      - 퓨샷 러닝은 맞춤형 서비스를 제공하는데 유용
- **퓨샷 프롬프팅(Few-shot prompting)**
  - 퓨샷 러닝 능력을 극대화하여 원하느 결과를 더 정확하게 얻을 수 있도록 도와줌
    - AI가 단순히 학습하는 것을 넘어 원하는 결과물을 섬세하게 집어내도록 도움
### 제로샷 프롬프팅
- AI에게 아무런 예시 없이 바로 원하는 결과를 요청하는 방법
- AI는 방대한 지식을 바탕으로 질문에 답
### 원샷과 투샷, 퓨샷 프롬프팅의 활용
- 원샷과 투샷 프롬프팅
  - 각각 하나 또는 두 개의 예시를 제공하여 원하는 결과를 유도하는 방식
  - 친구에게 새로운 게임 방법을 한두 번 시범 보이는 것
  - 제로샷보다 정확도 높음
  - 간단한 패턴 학습에 효과적
- 퓨샷 프롬프팅
  - 여러 개의 예시를 제공하여 원하는 결과 유도
  - 요리 선생님에게 여러 번 레시피를 배우는 것
  - 퓨샷 프롬프팅은 복잡한 패턴이나 규칙을 학습하고 다양한 맥락을 이해하는데 효과적

# 3. ChatGPT 앱과 나만의 맞춤형 GPT 만들기
## 1) ChatGPT 앱 활용
### ChatGPT 앱 프로그램
- 독립 실행형 앱: 브라우저 없이 실행 가능
- 빠른 접근성과 멀티태스킹 지원: Alt + Tab 전환 -> 업무 효율 향상
- 네이티브 기능 통합: 단축키, 다크 모드, 알림 기능 등 윈도우 시스템과 연동
- 성능 최적화: 브라우저 대비 메모리 사용 감소 및 안정적인 실행

## 2) GPT 탐색: 나만의 GPT 활용
### 개인 맞춤형 GPT
- 개요
  - 맞춤형 GPT는 사용자의 필요와 목적에 맞춰 개인화된 GPT
  - 맞춤형 GPT를 통해 특정 분야나 관심사에 적합한 기능 추가 가능
  - 사용는 자신의 요구에 맞춰 GPT의 성격, 스타일, 전문 영역 설정
