# 동아리 합격 이메일 자동 발송 가이드
> Zapier × Google Sheets × Gmail 자동화

---

## 개요

| 항목 | 내용 |
|------|------|
| **목적** | 합격자 명단에 "합격" 입력 시 Gmail 자동 발송 |
| **사용 도구** | Zapier, Google Sheets, Gmail |
| **예상 소요 시간** | 약 15~20분 |
| **필요 계정** | Zapier 계정, Google 계정 |

---

## Zap 설계 구조

```
🔵 Trigger (시작 조건)
   └── 앱: Google Sheets
   └── 이벤트: New or Updated Spreadsheet Row
   └── 조건: 합격자 명단 시트에 행이 추가/수정될 때

🔧 Filter (필터링)
   └── "합격여부" 열이 "합격"인 경우만 진행

🟢 Action (실행)
   └── 앱: Gmail
   └── 이벤트: Send Email
   └── 수신자: Sheets의 "이메일" 열
   └── 내용: 미리 작성한 고정 합격 안내 이메일
```

---

## 1단계. 스프레드시트 준비

Google 스프레드시트를 아래 형식으로 구성합니다.

| A: 이름 | B: 이메일 | C: 합격여부 |
|--------|----------|-----------|
| 홍길동 | hong@gmail.com | 합격 |
| 김철수 | kim@gmail.com | |
| 이영희 | lee@gmail.com | 합격 |

> ⚠️ **주의:** 1행은 반드시 헤더(열 제목)로 구성해야 합니다. Zapier가 열 이름을 자동 인식합니다.

---

## 2단계. Zapier에서 새 Zap 만들기

1. [zapier.com](https://zapier.com) 접속 후 로그인
2. 우측 상단 **[+ Create]** 클릭
3. **[Zaps]** 선택

---

## 3단계. Trigger 설정 (Google Sheets)

1. 앱 검색창에 **Google Sheets** 입력 후 선택
2. Event: **New or Updated Spreadsheet Row** 선택
3. **[Connect]** 클릭 → Google 계정 로그인 및 접근 허용
4. Spreadsheet: 합격자 명단 파일 선택
5. Worksheet: 해당 시트 탭 선택
6. **[Test trigger]** 클릭 → 샘플 데이터 확인

✅ **성공 기준:** 이름, 이메일, 합격여부 데이터가 화면에 표시되면 완료

---

## 4단계. Filter 추가 (합격자만 필터링)

"합격" 표시된 행에만 이메일이 발송되도록 필터를 추가합니다.

1. **[+]** 버튼 클릭 → **Filter** 선택
2. 조건 설정:
   - 필드: **합격여부 (C열)**
   - 조건: **(Text) Contains**
   - 값: `합격`
3. "Your Zap will proceed if..." 메시지 확인

✅ **성공 기준:** 조건 설정 후 초록색 체크 표시 확인

> ⚠️ **주의:** 값 입력 시 띄어쓰기, 오타에 주의하세요. `합격 ` (뒤에 공백)처럼 입력하면 작동하지 않습니다.

---

## 5단계. Action 설정 (Gmail 발송)

1. **[+]** 버튼 클릭 → **Gmail** 검색 후 선택
2. Event: **Send Email** 선택
3. Gmail 계정 연결

### 필드 매핑

| 필드 | 입력값 |
|------|--------|
| **To** | Trigger의 `이메일(B열)` 데이터 선택 |
| **Subject** | `[동아리명] 합격을 축하드립니다 🎉` |
| **Body Type** | Plain 또는 HTML 선택 |
| **Body** | 아래 이메일 본문 템플릿 붙여넣기 |

### 이메일 본문 템플릿

```
안녕하세요!

○○ 동아리에 지원해 주셔서 진심으로 감사드립니다.
심사 결과, 최종 합격을 알려드립니다.

---

📅 첫 모임 일정: 0월 0일(요일) 오후 0시
📍 장소: ○○관 ○○호

---

궁금한 점이 있으시면 언제든지 이 메일로 회신해 주세요.
함께하게 되어 진심으로 반갑습니다!

○○ 동아리 운영진 드림
```

> 💡 **Tip:** 이름을 개인화하고 싶다면 본문 첫 줄을 `안녕하세요, [Trigger의 이름 데이터]님!` 으로 변경하고, 입력창의 **+** 아이콘으로 Sheets의 이름 열을 삽입하세요.

4. **[Test action]** 클릭 → 본인 이메일로 테스트 발송 확인

✅ **성공 기준:** 본인 받은편지함에 테스트 이메일 수신 확인

---

## 6단계. 활성화

1. 우측 상단 **[Publish]** 클릭
2. Zap 이름 저장 (예: `동아리 합격 이메일 자동 발송`)
3. 토글이 **ON** 상태인지 확인

🎉 이제 스프레드시트 C열에 "합격"을 입력하면 자동으로 이메일이 발송됩니다!

---

## 자주 발생하는 오류와 해결법

| 증상 | 원인 | 해결법 |
|------|------|--------|
| Trigger 테스트에서 데이터가 안 보임 | 스프레드시트에 데이터가 없음 | 데이터를 최소 1행 이상 입력 후 재시도 |
| 이메일이 발송되지 않음 | Filter 조건 오타 | "합격여부" 값과 Filter 입력값 일치 여부 확인 |
| 모든 행에 이메일이 발송됨 | Filter 단계 누락 | 4단계 Filter를 다시 추가 |
| 계정 연결 오류 | 권한 미승인 | Google 계정 재연결 및 접근 권한 허용 |
| 무료 플랜 한도 초과 | 월 100회 실행 제한 초과 | Zapier Starter 플랜($19.99/월) 업그레이드 고려 |
